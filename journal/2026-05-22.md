## 10. SearXNG 搜索

**问题**：ingest 6 维度中 4 个依赖搜索，但 WebSearchAdapter 是空壳。

**方案**：
- 在服务器 localhost 上 Docker 部署 SearXNG（端口 8088）
- 配置 Google + Bing + DuckDuckGo + Wikipedia + GitHub 引擎
- WebSearchAdapter 新增 `_search_searxng()` 方法，调用 JSON API
- `auto` 模式优先级：searxng → zhipu → google → bing_cn
- `time_range` 参数移除（SearXNG 所有 time_range 值返回 0 结果）

**发现的问题**：
- SearXNG `time_range` 参数不可用（day/week/month/year 全返回 0 结果），需要在采集后按日期过滤
- 搜索质量取决于关键词："资本决策" 返回字典释义而非新闻

**commit**: `c779b98` feat(ingest): v1.2 早报 pipeline

**状态**：✅ 已完成

## 11. LLM 批量解读

**问题**：需要 LLM 对采集结果生成一句话解读 + Top 5 精选。

**方案**：
- 使用智谱 glm-5.1，走 Anthropic Messages API 协议（`https://open.bigmodel.cn/api/anthropic/v1/messages`）
- OpenAI-compatible 协议在 codingplan 上超时且返回空响应，Anthropic 协议快速可靠
- `interpret_dimension()`：批量解读，所有 Entity 一次 LLM 调用
- `generate_top5()`：四维分析（公司/战略/技术/启示）
- Prompt 模板中 JSON 花括号需转义为 `{{}}` 避免 Python `format()` 误解析

**commit**: `c779b98`

**状态**：✅ 已完成

## 17. 搜索关键词优化

**问题**：SearXNG 底层走 Bing 引擎，中文关键词搜索质量极差（返回字典页面、百科条目），开源趋势/应用落地 0 结果。

**发现**：
- `categories=news` 返回 0 结果（SearXNG news 引擎未启用）
- 中文查询 `"AI 开源框架 更新"` 返回开源中国等目录页
- 英文查询 `"open source AI model release 2026"` 返回高质量结果

**方案**：将 `.linglong.yaml` 中所有搜索关键词改为英文为主、中文为辅。

**效果对比**：
| 维度 | 优化前 | 优化后 |
|------|--------|--------|
| 开源趋势 | 0 条 | 9 条 |
| 公司决策 | ~9 条（低质量） | 51 条 |
| 资本决策 | 字典页面 | 10+ 条新闻 |

**文档更新**：3 份文档的示例关键词同步更新 + 设计总览加搜索建议。

**commit**: `5d27e7d` docs(ingest): 关键词示例更新为英文 + 搜索建议

**状态**：✅ 已完成
